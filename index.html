Prestech Brasil

A Prestech é uma consultoria que surgiu em Setembro de 2017 com dois amigos que possuiam vasta experiência no mundo da programação, um, Victor com experiência em desenvolvimento Full-Stack nas linguagens Javascript,Typescript, utilizando diversos Frameworks Javascripts. O Outro Gabriel com vasta experiência em Dev-ops utilizando principalmente a Cloud da Amazon, a AWS, mas com experiências sólidas em Azure, Google Cloud, Digital Ocean entre outros.

Serviços

Banco de dados

Utilização de bancos de dados SQL e também bancos de dados NoSQL, além de utilizar estes, é feito uma análise do projeto de sua empresa para melhor escolher o banco de dados ideal para performar suas aplicações

Desenvolvimento Frontend
